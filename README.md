## Quick start

Quick start options:

- `npm i`
- `npm start`

#Deployment

    Deployed in:

- localhost:3000 - Login Page
- localhost:3000/admin - page containing dashboard and other main pages.
- localhost:3000/rtl - Core carrier Page and its sub pages.
